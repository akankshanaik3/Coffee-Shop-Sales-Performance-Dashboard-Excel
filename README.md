
# ☕ Coffee Shop Sales Performance Dashboard (Excel)

## Overview

This project is an interactive Coffee Shop Sales Performance Dashboard built using Microsoft Excel. It provides key business insights through dynamic dashboards, KPI cards, Pivot Tables, Pivot Charts, and Slicers.
